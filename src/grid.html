grid

grid - это двухмерная система компоновки блоков на основе сетки.

grid объявляется в контейнере
.grid-container
задается с помощью свойства
display: grid;
далее задаются кол-во колонок и столбцов, а так же их размеры
grid-tamplate-columns: 100px 100px 100px;
grid-tamplate-rows: 50px 50px 50px;
для сокращения кода можно записать в одну строчку
grid-tamplate: 50px 50px 50px / 100px 100pz 100px
потом задаем расстояние между блоками
columns-gap: 20px;
rows-gap: 20px
если расстояние одинаковое можно записать
gap: 20px;

fr - единица измерения , которая делит на равные части.

repeat() - для компактной записи повторяющихся столбщов и линеек.
пример: grid-tamplate-columns: repeat(3 (сколько раз будем повторять), fr (что повторять));


в явных гридах (которые не поместились в сетку) то же задаются размеры
grid-auto-columns: 200px;

если объем текста меняется, для изменения размера блока используется (а так же при одоптации):
minmax()
repeat(3, minmax(200px, 300px (auto))

для заполнения элементами всей ширины контейнера используется:
auto-fit
grid-tamplate-columns: repeat(auto-fit,minmax(100px, 300px))

auto-fill - оставляет место под элеменыт.

grid-auto-flow - свойтво управляет поведение автоматически размещенных элементов
grid-auto-flow: colomn;

позиционирование треков делают с помощью свойств:
grid-colomn-start: 3;
grid-colomn-end: 5;
grid-row-start: 1;
grid-row-end: 3;
можно сократить:
grid-colomn: 3/5;
grid-row: 1/3;
