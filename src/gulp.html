установка и настройка gulp

при первой установке на компьютер сначало устанавливаем nodejs

nodejs

1. создаем проект для наших пакетов :
npm init
(в водим в терминал), создается файл package.json


далее в водим команду:
npm install --global gulp-cli
(команда позваляющая работать с терминалом)


устанавливаем gulp:
npm install --save-dev gulp


устанавливаем компилятор :
npm i gulp-sass --save-dev


устанавливаем :
npm install i browser-sync --save-dev


устанавливаем :
npm i gulp-autoprefixer --save-dev


устанавливаем :
npm i gulp-clean-css --save-dev


создаем файл для настроек в проекте :
gulpfile.js


устанавливаем :
npm i gulp-rename --save-dev
