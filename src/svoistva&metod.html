свойства строк

.length - свойства для измерения длины строки.

методы строк

.toUpperCase() - метод изменяющий строку в верхний регистр

.toLowerCase() - метод изменяет строку в нижний регистр.

.indexOf() - поиск в строке искомое значение (в скобках используются аргументы)

.slice() - метод используется для вырезания слова ,в скобках используются аргументы , с кокого символа до какого.Либо один символ , с какого символа и до конца.

.substr() - метод похож на .slice() различие в аргументах, песрвый с какого символа , а второй сколько символов вырезать.

методы чисел.

Math.round() - используется для округления числа, в скобках ставится число ( переменная) которое надо округлить.

parseInt() - метод превращает строку в другой метод исчисления.

parseFloat() - метод превращает строку или число в десятичную.
