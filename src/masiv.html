Методы для работы с массивом.

.pop() - метод который удаляет последний элемент из массива.

.push() - метод который добавляет элемент в конец массива, в скобках указывается что добавляем.

.forEach() - метод испольется для перебора массива, его можно гибко настроить (задать функцию в скобках , которая будет срабатывать в каждом элементе массива.
пример:
arr.forEach(function(item, i , arr){ console.log(`${i} : ${item} - ${arr}`); })
item - содержимое массива
i - номер массива
arr - массив

.split() - метод который разбивает на массив строк путем разделением строки указанной подстракой.

.join() - метод который объединяет все элементы массива в строку.

.sort() - метод сортировки массива.

Object.assign() - метод используется для копирования всех собственных перечисленных свойств из одного или более исходных объектов в целевой.

... - оператор разварота используется для соединения нескольки массивов.
