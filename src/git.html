работа с git-ом

иницифлизируем проект:
git init


вводим имя и мочту:
git config --global user.name"sergey"
git config --global user.email"...@gmail.com"


добавляем файлы:
git add src


проверить статус:
git status


создать контрольную точку:
git commit -a -m"first"


создать на guthub репозиторий, взять команду с сылкой и ввести в терминал.


отправить на удаленный репозиторий:
git push
